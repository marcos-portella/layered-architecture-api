## JavaScript & Web Interaction

Este repositório é dedicado aos meus estudos de JavaScript, desenvolvidos em paralelo com a minha graduação em **Análise e Desenvolvimento de Sistemas (ADS)**. O foco aqui é dominar a linguagem que dá vida à web e aprender a conectar interfaces modernas ao meu ecossistema de APIs.

### Foco Acadêmico e Prático:

- **Fundamentos de JS**: Variáveis (let/const), tipos de dados, operadores e estruturas lógicas.

- **Manipulação de DOM**: Aprendendo a interagir com elementos HTML para criar páginas dinâmicas.

- **Consumo de APIs (Fetch API)**: Integrando o Frontend com meus backends em **Python (FastAPI)** e **Java**, utilizando o **ngrok** para testes em tempo real.

- **Asincronismo**: Domínio de ``async/await`` e ``Promises`` para garantir uma experiência de usuário fluida.

### Stack de Estudo:

- **Linguagem**: JavaScript (ES6+)

- **Ambiente**: Navegador (DevTools) & Node.js

- **Integração**: Fetch API & JSON

Estudar JavaScript na faculdade me permite fechar o ciclo do desenvolvimento, transformando dados complexos de faturamento em interfaces visuais e intuitivas para o usuário final.
