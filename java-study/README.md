## Java Development & Software Foundations

Este repositório é dedicado ao meu aprofundamento na linguagem **Java** e nos princípios da **Programação Orientada a Objetos (POO)**. Aqui, documento minha evolução técnica para além do Python, focando em sistemas que exigem alta tipagem e performance.

### Objetivos de Estudo:

- **Core Java**: Domínio de tipos primitivos, estruturas de controle e manipulação de arrays.

- **POO**: Implementação de classes, construtores, herança e polimorfismo.

- **Robustez**: Estudo de tratamento de exceções e segurança de tipos.

- **JVM Ecosystem**: Entendimento de Bytecode, compilação e gerenciamento de memória.

### Tecnologias & Ferramentas:

- **Linguagem**: Java (JDK 17+)

- **IDE**: IntelliJ IDEA / VS Code

- **Build Tools**: Maven (em breve)

A transição para Java me permite aplicar o rigor técnico da engenharia de software em um ecossistema robusto e amplamente utilizado em sistemas críticos.
