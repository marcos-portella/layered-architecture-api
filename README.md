## API de Gestão de Clientes (FastAPI + MySQL)

Este projeto é uma API RESTful completa para gerenciamento de clientes, focada em arquitetura modular e boas práticas de desenvolvimento.

### Tecnologias Utilizadas:
* Python 3.13 / FastAPI
* MySQL (Banco de Dados Relacional)
* Pytest (Testes de Integração)
* Pydantic (Validação de Dados)
